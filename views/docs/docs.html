AngularJS - Login Application-PHP web services

Native stand-alone application make a angularJs services based on Ngcookies serving access permissions from Server

Features:

  • Validate session
  • Authenticate Pages
  • Destroy session and clear data
$http request to login
Method
authenticate.login(data,loginUrl, callback);

data: json structure to send server eg.
{username:"mahendra","password":"123456","remember":true}

loginUrl: request url eg. http://localhost/webservices/login

callback : function() received data form seerver side

authenticate.login(data,loginUrl,
function(response){ if(response.valid==true)
{
//if username and password is valid
}
else if(response.valid==false)
{
//if username or password is invalid
}
})

Set session data on local system

Method

authenticate.setAuthenticatedUser(response);

response: json structure eg.
{valid:"true",key:"121212121",username:"mahendra.singh"}

Check session on local system

Method

authenticate.validateSession()

this function return true if valid session otherwise return false

Page authorization User don't access page without login

Method

define array
pages=['/services',"/"]
authenticate.pageAuth(currentPath,loginPagePth)

currentPath: set the current page location
loginPath : set login page path if you want to redirect

Page authorization User allow to access page without login

Method

define array
pages=['/abouts',"/docs"]
authenticate.pageAuthInvalid(currentPath,redirect);

currentPath: set the current page location
redirect : if you want to redirect and other page

Logout Clear data or session

Method

authenticate.removeAuthenticatedUser();

remove all data form local session
